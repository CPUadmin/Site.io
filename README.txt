- index.html
- services.html
- папку assets
- файл resume.pdf

Как загрузить:
1. Распакуй архив
2. Загрузи все файлы в корень сайта на хостинге
3. Убедись, что index.html лежит в корне

- сайт статический, backend не нужен
- переключение темы работает через localStorage
- переключение языка RU/EN работает на фронте
