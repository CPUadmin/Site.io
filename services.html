Услуги

Форматы работы, в которых я могу быть полезен

Без пафоса и без расплывчатости. Ниже три нормальных сценария: сопровождение, проектная работа и разовые задачи.

Сопровождение офиса

Когда нужен человек, который держит инфраструктуру в порядке

  • Поддержка рабочих мест, доступов, серверов и сетевой части
  • Удалённый доступ, мониторинг, резервные копии, типовые инциденты
  • Документация и нормализация того, что раньше было хаосом
Проект

Когда нужно развернуть, перенести или аккуратно внедрить что-то новое

  • Развёртывание сервисов и серверной части
  • Миграции, VPN, виртуализация, схема доступов
  • Инструкции, шаблоны и запуск пользователей
Разовые задачи

Когда нужна точечная помощь без полноценного сопровождения

  • Разобрать проблему с сетью, сервером или доступами
  • Настроить мониторинг, скрипт, VPN или рабочую схему
  • Быстро включиться, стабилизировать и объяснить, что дальше
Обычно входит
  • Windows Server / Linux
  • Active Directory / учётные записи / доступы
  • MikroTik / VPN / WireGuard / VLAN
  • VMware / Hyper-V / Proxmox
  • Zabbix / алерты / логирование
  • PowerShell / Python / документация
По умолчанию не входит
  • Непрофильная рутина вне договорённости
  • Бухгалтерские и административные задачи
  • Неограниченный объём работ без понятной зоны ответственности
Контакт

Можно прийти с проектом, с проблемой или просто с вопросом

Если задача вменяемая и у неё есть нормальные рамки, дальше уже разберёмся.
